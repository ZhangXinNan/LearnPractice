
# 区别

区别　|　静态库　| 动态库
----|----|----
载入时刻　| 编译时已经被载入 |　运行时载入内存，编译过程中仅简单的引用
共用　| |　不同程序调用相同的库，内存只只需要一份该动态库实例
最大区别　|　把库直接加载到程序中 | 动态库链接时，只是保留接口，将动态库与程序代码独立。
编译时　| 连接到目标代码中 |　并不连接到目标代码中
运行时　| 程序运行时不再需要该静态库 | 程序运行才被载入，运行时还需要动态库存在
