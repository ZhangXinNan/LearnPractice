左值就是放在等号左边的值；右值就是放在等号右边的值。

左值引用 就是左值的引用；右值引用 就是右值的引用 。



### lvalue  纯左值
有身份的对象，可以持续找到它

### prvalue 纯右值
表示一个没有身份、只关心值的临时结果

通常没有地址

### xvalue  将亡值
1. 返回右值引用的函数，值类型是`xvalue`。
2. `std::move(v)`除了返回v的右值引用之外，不会做任何其他的事情。
3. `xvalue` 本质上是左值，只是贴上了一个“语义”标签的左值。
4. 右值引用是可以绑定到`xvalue`或`prvalue`的引用；左值引用必须绑定到`lvalue`的引用。



构造函数的调用者 ->
通过std::move(a)作出承诺 ->
a的资源我不要了，你可以随意移动；

构造函数的实现者    ->
看到参数是右值引用  ->
放心地把资源移走；
