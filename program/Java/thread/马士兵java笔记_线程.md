
# 线程的使用
1. 继承Thread类，实现run()方法。调用start()启动线程。
2. 实现Runnable接口，实现run()方法，然后用Thread类实例化，调用 start() 启动线程。

# 线程状态转换
```
                阻塞状态-------
                    |         |
                阻塞解除   导致阻塞的事件
                    |         |
创建-->start()-->就绪状态<-->运行状态-->终止
                    
```
