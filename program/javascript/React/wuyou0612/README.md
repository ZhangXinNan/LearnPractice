# 1 项目搭建
首先创建一个基于React的项目。
第一种方式，引入一个文件。
第二种方式，使用脚手架。更常用。

关键步骤：
```bash
# 1 打开终端

# 2 npx create-react-app 项目名称
npx create-react-app my-react-app

# 3 进入项目目录
cd my-react-app

# 4 启动项目
npm start

```

- public 页面资源
- src
  - index.js 入口文件
  - App.js
