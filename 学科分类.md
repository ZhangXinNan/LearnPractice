# 数学（Mathematics）细分

* **Pure Mathematics 纯数学**

  * Algebra 代数（群/环/域 Group/Ring/Field）
  * Number Theory 数论
  * Geometry & Topology 几何与拓扑
  * Real/Complex Analysis 实变/复分析
  * Functional Analysis 泛函分析
  * Logic & Set Theory 数理逻辑与集合论
* **Applied Mathematics 应用数学**

  * ODE/PDE 常微/偏微分方程
  * Numerical Analysis 数值分析（FEM/Finite Difference）
  * Optimization 优化（Convex/Nonconvex）
  * Dynamical Systems 动力系统
  * Mathematical Physics 数学物理
* **Probability & Statistics 概率与统计**

  * Probability Theory 概率论
  * Statistical Inference 统计推断
  * Bayesian Statistics 贝叶斯统计
  * Time Series 时间序列
  * Machine Learning 统计学习
