
# 1. 由两点求直线参数
直线公式：
$$ Ax + By + C = 0 $$

假如两个点代入直线公式：
$$ Ax_1 + By_1 + C = 0 $$
$$ Ax_2 + By_2 + C = 0 $$
两式相减得到：
$$ A(x_1-x_2) + B(y_1-y_2)=0 $$

 如果令 $A = y_1 - y_2$，那么$B = x_2 - x_1$
 代入得：
 $$
 x_1(y_1-y_2) + y_1(x_2-x_1) + C = 0
 $$
 $$
 x_1y_1 - x_1y_2 + x_2y_1 - x_1y_1 + C = 0
 $$
 所以：
 $$
 C = x_1y_2 - x_2y_1
 $$


# 2. 由两直线求交点

两直线公式为：
$$ A_1x + B_1y + C_1 = 0 $$
$$ A_2x + B_2y + C_2 = 0 $$

解得交点坐标：
$$ x = \frac{B_1C_2 - B_2C_1}{A_1B_2-A_2B_1} $$
$$ y = \frac{A_2C_1-A_1C_2} {A_1B_2-A_2B_1}$$
