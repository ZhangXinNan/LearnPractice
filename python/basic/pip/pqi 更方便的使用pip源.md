

1. 安装pqi
```bash
pip install pqi
```

2. 列举支持的pip源
```bash
pqi ls
```
3. 改变pip源
```bash
pqi use douban
```

4. 显示当前pip源
```bash
pqi show
```

5. 添加源
```bash
pqi add ustc https://mirrors.ustc.edu.cn/pypi/web/simple
```

6. 移除源
```bash
pqi remove pypi
```
