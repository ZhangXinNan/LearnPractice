## 5.3 负载均衡
Elastic Load Balancing (ELB)
- 向EC2实例的HTTP/HTTPS/TCP流量分发请求，实现负载均衡
- 支持运行状况检查，帮助清除替换不健康的实例
- 提供流量，延迟信息，帮助动态扩展和缩减所需的资源


- 创建Classic负载均衡器
  - 步骤1：定义负载均衡器
  - 步骤2：分配安全组
  - 步骤3：配置安全设置
  - 步骤4：配置运行状况检查
  - 步骤5：添加EC2实例
  - 步骤6：添加标签
  - 步骤7：审核
