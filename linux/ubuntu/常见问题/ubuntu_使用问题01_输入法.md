
在Ubuntu中安装并使用输入法
_____________________________

```sh
sudo apt-get install ibus-table-wubi ibus-pinyin
# ubuntu20.04
sudo apt install fcitx-table-wubi
```

然后重启。

重启后Text Entry Settings里添加上需要的输入法。

一般将快捷键设置为 Ctrl + Space
