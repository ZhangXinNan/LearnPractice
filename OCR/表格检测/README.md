
# 一、研究背景 
文档页面往往由不同的页面对象（Page Object）组成，包括文本行、图像、表格、公式等。

**表格检测（Table Detection）**任务是从一个页面中检测出表格所在的区域，
**表格结构识别（Table Structure Recognition）**任务则是在检测到的表格区域的基础上，进一步将表格的内容与逻辑结构识别出来。
