

# 4、中文文档文字识别

## 4.1 PP-OCRv4_mobile_rec
```bash
python tools/eval.py -c configs/rec/PP-OCRv4/PP-OCRv4_mobile_rec.yml \
    -o Global.pretrained_model=output/PP-OCRv4_mobile_rec_pretrained.pdparams \
    Global.use_gpu=True \
    Eval.dataset.data_dir=/home/zhangxin/data_public/OCR/4_Chinese-Document-Text-Recognition \
    Eval.dataset.label_file_list='[/home/zhangxin/data_public/OCR/4_Chinese-Document-Text-Recognition/train_1000.txt]'
```
